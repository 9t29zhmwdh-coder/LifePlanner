# Roadmap — LifePlanner

- [x] Text/email/PDF parser for events, tasks, deadlines
- [x] Intelligent event-task linking
- [x] CalDAV sync (local network)
- [x] SQLite FTS5 storage
- [x] AI suggestions via Ollama
- [x] Tauri v2 desktop shell
